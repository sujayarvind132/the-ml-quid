Quick Setup Guide

  1. Create a new repository on GitHub with any name (e.g., "ai-search-assistant")
  2. Upload these files to your repository:
    • index.html
    • style.css
    • app.js
  3. Enable GitHub Pages:
    • Go to repository Settings
    • Scroll to "Pages" section
    • Select "Deploy from a branch"
    • Choose "main" branch and "/ (root)" folder
    • Click "Save"
  4. Access your site at: https://yourusername.github.io/repository-name

Note: This app uses mock data for demonstration. In a production environment, you would integrate with real search APIs and AI services.
